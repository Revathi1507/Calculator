# Simple Calculator in Java

This project is a **console-based Simple Calculator** developed using **Java**.  
It is designed especially for **beginners** to understand basic Java concepts such as input handling, switch-case, loops, and operators.

---

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Modulus (%)
- Handles **invalid operator input**
- Repeats operation selection until a valid operator is entered
